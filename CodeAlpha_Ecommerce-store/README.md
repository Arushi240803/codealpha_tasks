# Full Stack E-commerce Store

A full stack e-commerce web application built using Node.js, Express.js, MongoDB, HTML, CSS, and Vanilla JavaScript.

This project includes user authentication, product listing, shopping cart functionality, product details page, and order processing.

---

# Tech Stack

## Frontend

* HTML
* CSS
* Vanilla JavaScript

## Backend

* Node.js
* Express.js

## Database

* MongoDB

## Other Tools & Packages

* Mongoose
* CORS
* dotenv
* Nodemon

---

# Features

* User Registration & Login
* Product Listing
* Product Details Page
* Shopping Cart
* Order Processing

# Installation & Setup

## Clone Repository

git clone https://github.com/Arushi240803/Ecommerce-Store-.git

---

# Backend Setup

## Move to backend folder

cd backend

## Install dependencies

npm install

## Run backend server

npm run dev

Server runs on:

http://localhost:5000

---

# Frontend Setup

Open frontend HTML files using Live Server extension in VS Code.

Example:

* Right click on index.html
* Click "Open with Live Server"
